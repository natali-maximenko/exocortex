### Результаты месяца

> Результат = измеримое состояние конкретной **системы** по итогам месяца. Может быть абстрактным (метрика, характеристика), но должен быть верифицируем.
> Связь: НЭП → R → Q-цель квартала. Подробно: PD.SOTA.007-strategy-hierarchy (PACK-personal).
> **Источник:** каждый R привязан к S-НЭП (триада Проблема + Неудовлетворённость + Эмоция, [Dissatisfactions.md](Dissatisfactions.md)). R без привязки к S-НЭП → вопрос «зачем».

| ID | Система | Результат | НЭП | → Q# | Критерий done | Бюджет |
|----|---------|-----------|-----|------|---------------|--------|
| R1 | ... | ... | ... | Q{N}.1 | ... | ...h |

**Бюджет месяца:** ~{N}h РП всего / ~{N}h на R1-R{N} / ~{N}h физ / Плановый мультипликатор ~{N}x
